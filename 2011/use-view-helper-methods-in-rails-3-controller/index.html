Use View Helper Methods in Rails 3 Controller

09 September 2011 on Rails

Sometimes people use helper_method to turn their controller methods to be able to use in their views. Rails provides a lot of view helpers, and sometimes we would like to use some helper methods that have in helper modules in our controllers instead, for example link_to, mail_to, pluralize, etc. One way to archive this in Rails 3 is to call view_context inside the controller to create a new ActionView instance for a controller, then all helper methods will be available through this instance in the controller.
